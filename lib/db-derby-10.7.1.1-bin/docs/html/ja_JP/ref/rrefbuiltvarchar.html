VARCHAR関数

VARCHAR関数は文字列を表す、可変長文字列を返します。

文字列を可変長文字列に変換する構文

VARCHAR (文字列式 )
文字列式
式の値は最大で32,672バイトの文字列でなければなりません。

日時を可変長文字列に変換する構文

VARCHAR (日時式 )
日時式
式の値は、日付、時刻、タイムスタンプの何れかの型でなければなりません。
EMPLOYEE表より、"Dolores Quintana"の仕事の説明(CHAR(8)のJOB)を、可変長文字列の値として選択する。 
SELECT VARCHAR(JOB)
FROM EMPLOYEE
WHERE LASTNAME = 'QUINTANA'
