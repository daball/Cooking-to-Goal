時刻書式のJDBCエスケープ構文

Derbyは時刻のJDBCエスケープ構文を、等価な時刻のSQL構文に解釈します。Derbyでは8文字(6桁と小数点以下2桁)のISO書式も利用できます。

構文

{t 'hh:mm:ss'}

次と等価です。

TIME 'hh:mm:ss'

例

VALUES {t '20:00:03'}
