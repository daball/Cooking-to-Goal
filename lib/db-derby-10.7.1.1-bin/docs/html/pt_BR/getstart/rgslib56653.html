Bibliotecas não fornecidas pelo Derby

No ambiente Java Development Kit Versão 1.3 algumas funcionalidades especiais do Derby requerem que sejam instaladas bibliotecas adicionais, e que estas sejam colocadas no caminho de classe (O Java Development Kit Versão 1.4 já inclui todas estas bibliotecas).

  • LDAP (consulte a publicação Guia do Desenvolvedor do Derby)
  • JTA (consulte a publicação Manual de Referência do Derby)
  • JDBC 2.0 Extensions (consulte a publicação Manual de Referência do Derby)
